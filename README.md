# AI Ops Monitoring with Anomaly Detection

基于 Python + Machine Learning 的 AI 运维异常检测系统。

## Features

- Linux 服务器指标采集
- CPU / Memory 监控
- Isolation Forest 异常检测
- AI Ops 可视化分析
- 异常聚类识别
- 无监督机器学习检测
